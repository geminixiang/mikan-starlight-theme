## 提示框

五種 aside 變體，以及自訂標題：

:::note
Note aside：預設的資訊提示。
:::

:::tip
Tip aside：適合有幫助但非必要的建議。
:::

:::caution
Caution aside：暗色模式會分開背景、表面與邊框層，避免看起來混濁。
:::

:::danger
Danger aside：最高強度的警告。
:::

:::note[自訂標題]
這個 aside 使用自訂標題，而不是預設標籤。
:::

## 步驟

<Steps>

1. 在 Astro Starlight 專案中安裝 `@geminixiang/mikan-starlight-theme`。

2. 在設定中註冊 `mikanTheme()` 與 `mikanAgentMarkdown()`。

3. 設定程式碼主題：

   ```js
   markdown: {
     shikiConfig: {
       theme: mikanCodeTheme;
     }
   }
   ```

4. 啟動開發伺服器並檢查這些測試頁。

</Steps>

## 表格

| Token           | 亮色      | 暗色      | 用途                 |
| --------------- | --------- | --------- | -------------------- |
| `--mk-bg`       | `#ffffff` | `#0a0a0a` | 頁面 canvas          |
| `--mk-surface`  | `#ffffff` | `#191919` | 卡片填色             |
| `--mk-ink-fill` | `#0a0a0a` | `#ffffff` | 唯一的填色膠囊按鈕   |
| `--mk-accent`   | `#2563eb` | `#a0c3ec` | 安靜連結 / active 狀態 |
