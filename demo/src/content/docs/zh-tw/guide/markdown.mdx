## 文字與強調

一般內文包含 **粗體文字**、_斜體文字_、***粗斜體***、~~刪除線~~ 與 `inline code`。這裡也放一個[一般連結](https://astro.build/)與一個[外部連結](https://starlight.astro.build/)，確認強調色安靜但清楚可讀。

這是一段較長的文字，用來檢查閱讀寬度與行高。Mikan 追求平靜且緊實的閱讀節奏：內文尺寸舒適、行高充足；標題則用較緊的負字距，讓階層看起來精準收束，而不是大聲喧嘩。

## 引言

> 引言應該像安靜的旁註：起始側一條細線、低調文字，沒有背景填色。
>
> — 同一段引言中的第二段，用來確認間距。

## Inline code 與按鍵

按 <kbd>⌘</kbd> + <kbd>K</kbd> 開啟搜尋。像 `pnpm dev`、`--mk-accent` 與 `npx getdesign add x.ai` 這類 inline code 應該保持可讀，但不搶走正文注意力。
