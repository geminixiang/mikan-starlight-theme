The `mikanCodeTheme` is a low-chroma editorial palette: a single blue reserved
for keywords, green strings, violet constants, and muted comments. These
samples confirm the token colors hold up across languages, in both light and
dark mode.

## Syntax across languages

```ts title="theme.ts"
import type { StarlightPlugin } from "./types";

export function mikanTheme(options: { dark?: boolean } = {}): StarlightPlugin {
  const { dark = true } = options;
  return {
    name: "starlight-theme-mikan",
    hooks: { "config:setup": () => console.log(`dark=${dark}`) },
  };
}
```

```python title="hello.py"
from dataclasses import dataclass

@dataclass
class Theme:
    name: str
    accent: str = "#2563eb"

    def describe(self) -> str:
        return f"{self.name} accent={self.accent}"

print(Theme("mikan").describe())
```

```rust title="main.rs"
#[derive(Debug)]
struct Theme<'a> {
    name: &'a str,
    accent: u32,
}

fn main() {
    let theme = Theme { name: "mikan", accent: 0x2563eb };
    println!("{theme:?}");
}
```

## Expressive Code features

A diff with inserted, deleted, and highlighted lines:

```js title="config.mjs" ins={4} del={3} {6}
export default {
  integrations: [
    mikanTheme({ accent: "orange" }),
    mikanTheme({ accent: "ink" }),
  ],
  markdown: { shikiConfig: { theme: mikanCodeTheme } },
};
```

A terminal frame (no language id, `sh`/`bash` renders the terminal chrome):

```bash
git clone https://github.com/geminixiang/mikan-starlight-theme
cd mikan-starlight-theme
pnpm install && pnpm dev
```

Line numbers and a marked range:

```ts showLineNumbers {2-3}
const themes = ["mikan-light", "mikan-dark"];
const light = themes[0];
const dark = themes[1];
export { light, dark };
```
