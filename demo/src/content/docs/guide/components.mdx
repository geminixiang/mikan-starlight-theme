Starlight's built-in components inherit the mikan visual language — hairline
borders, 8px cards, no shadows.

## Admonitions

All five aside variants, including a custom title:

:::note
A note aside — the default informational callout.
:::

:::tip
A tip aside — for helpful, optional guidance.
:::

:::caution
A caution aside — dark mode separates background, surface, and border layers so
nothing reads as muddy gray.
:::

:::danger
A danger aside — the highest-emphasis warning.
:::

:::note[Custom title]
An aside with a custom title instead of the default label.
:::

## Steps

<Steps>

1. Install `@geminixiang/mikan-starlight-theme` next to Astro Starlight.

2. Register `mikanTheme()` and `mikanAgentMarkdown()` in your config.

3. Set the code theme:

   ```js
   markdown: {
     shikiConfig: {
       theme: mikanCodeTheme;
     }
   }
   ```

4. Run the dev server and verify these fixtures.

</Steps>
